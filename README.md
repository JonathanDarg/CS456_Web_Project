# CS456_Web_Project

<h3>For windows devices</h3>
to run our static html page, visit the view/WebPage/src directory and run the command ``` start .\index.htm ``` in the terminal. This will open the  stand-alone full HTML file in your default browser.

<h3>For Mac devices</h3>
Just like windows, navigate to the view/WebPage/src directory and run the command ``` open index.htm ``` in the terminal. This will open the stand-alone full HTML file in your default browser. <br></br> 

just open the index.html file in the browser of your choice, and you should be able to see the webpage.
